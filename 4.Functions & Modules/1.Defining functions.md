## ðŸ”¹ Defining Functions

Functions are blocks of code that perform a specific task. You define a function using the `def` keyword.

```python

### ðŸ”¸ Example: Function with No Parameters

### ðŸ”¸ Example: Function with Parameters

```python

    return a + b

### ðŸ”¸ Example: Function with Default Parameters

```python

    print(f"Hello, {name}!")

```

```python

## ðŸ§± Organizing Code with Modules

A **module** is a `.py` file that contains functions, variables, or classes. You can import and reuse them in other files.

### ðŸ”¸ Creating a Module

Create a file called `my_module.py`:

```python
# my_module.py

def greet(name):
    return f"Hello, {name}!"
```

### ðŸ”¸ Importing and Using a Module

```python
# main.py

import my_module

print(my_module.greet("Alice"))
```

You can also import specific functions:

```python
from my_module import greet

print(greet("Bob"))
```
